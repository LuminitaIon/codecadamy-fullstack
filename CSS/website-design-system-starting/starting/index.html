The 'Sun Orange'colour is to be used around the site as the companies signature colour. It can be used to colour element backgrounds, shading and formatting text. It is not to be used for text due to resolution issues.

'Bat Black' can be used all over the web site for text, headings and borders

The 'Strong Tea Brown' colour is to be used for the main site name and any website contact details.

'Deep Ocean Blue' is to be used for footer information and signatures

The 'Burnt Grey' colour is to be used for page backgrounds in order for text to be easily read by clients

Text Styles

H1: Main Page Heading

  • Should only be used once.
  • Font Size: 72px or higher
  • Font Family: 'Assistent'
  • Font Weight: Bold 700
  • Colour: either #1DA4A8, #A84D02 or #000000

H2: Subheadings

  • To be used as subheading for new concept.
  • Font Size: 40px
  • Font Family: 'Assistant' or 'Catamaran'
  • Font must match its relevant heading/content
  • Font Weight: Regular 400
  • Colour: either #1DA4A8, #A84D02 or #000000

H3: Headings

  • To be used as brief description for H2 headings.
  • Font Size: 30px
  • Font Family: 'Assistant' or 'Catamaran'
  • Font must match its relevant heading/content
  • Font Weight: Regular 400
  • Colour: either #1DA4A8, #A84D02 or #000000

H4: Headings

  • Column headings and chapter titles.
  • Font Size: 25px
  • Font Family: 'Assistant' or 'Catamaran'
  • Font must match its relevant heading/content
  • Font Weight: Regular 400
  • Colour: either #1DA4A8, #A84D02 or #000000

Paragraph Elements

  • Font Size: 20px
  • Font Family: 'Assistant' or 'Catamaran'
  • Font must match its relevant heading
  • Font Weight: Regular 400
  • Colour: either #1DA4A8, #A84D02 or #000000
